Mis prácticas de la asignatura Laboratorio de Tecnologías audiovisuales en la Web. Curso 2019-2020

## Práctica 1:

Se trata del servidor de una tienda, hecha en nodejs, de una manera muy sencilla. Y cutre.

## Práctica 2:

## Práctica 3:

## Práctica 4:
